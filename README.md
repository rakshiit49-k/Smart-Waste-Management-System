Smart Waste Management System

Overview

The Smart Waste Management System is a console-based application developed in C. It simulates a basic waste management process by allowing users to manage waste bins, monitor their status, and perform waste-related operations through an interactive menu.

Features

- Add new waste bins
- View all waste bins
- Update bin details
- Search for a bin
- Delete a bin
- User-friendly menu-driven interface
- File handling for data storage (if implemented)

Technologies Used

- C Programming
- File Handling
- Structures
- Functions

How to Run

1. Clone the repository.
2. Open the project in your preferred C IDE or code editor.
3. Compile the source code using a C compiler (such as GCC).
4. Run the generated executable.

Project Structure

- "main.c" – Main program
- Other ".c" and ".h" files – Project modules and functions

Future Improvements

- Graphical User Interface (GUI)
- Database integration
- Real-time waste monitoring using IoT sensors
- Admin and user login system
